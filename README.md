This project presents a SQL-based Cab Booking System designed to manage customers, drivers, vehicles, bookings, rides, and payment-related data in a structured relational database. The project focuses on database schema design, normalization, data integrity, and efficient data retrieval using SQL and MySQL. It includes analytical and operational queries, joins, subqueries, aggregate functions, stored procedures, indexing, and role-based access concepts to demonstrate practical database management and SQL problem-solving skills.
