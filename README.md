# Cab Booking Management System – SQL Database Project

## Overview

This project presents a SQL-based Cab Booking Management System designed to manage customers, drivers, vehicles, bookings, rides, and payment-related data in a structured relational database.

The project focuses on relational database design, normalization, data integrity, SQL query development, stored procedures, indexing, transaction management, and role-based database access. It also includes analytical SQL queries to extract meaningful operational and business insights from cab booking data.

## Business Problem

A cab booking business needs an efficient database system to manage customers, drivers, vehicles, bookings, rides, and payments while maintaining data accuracy and consistency.

Without a properly structured relational database, the business may face challenges such as duplicate data, inconsistent records, inefficient data retrieval, poor transaction management, and difficulty generating useful business insights.

The objective of this project is to design and implement a structured SQL database system that supports daily cab booking operations while also enabling analytical reporting and data-driven decision-making.
